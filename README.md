# ES Utils

A Chrome extension that adds useful utilities for browsing the web.

## Features

### Image Utilities

Download and copy image URLs with overlay buttons that appear when you hover over images.

- Download images directly to your computer
- Copy optimized image URLs to clipboard
- Automatically extracts real URLs from Google Images, Google Drive, Dropbox, and Pinterest
- Works with lazy-loaded images

### Dark Mode

Toggle dark mode on any website with adjustable brightness and contrast.

- Simple on/off toggle
- Brightness control (50-150%)
- Contrast control (50-150%)
- Images remain in their original colors
- Settings persist across page reloads

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder

## Usage

Click the extension icon in your toolbar to access settings.

### Image Buttons

Buttons automatically appear on images larger than 15% of your screen size. You can enable or disable the download and copy buttons separately.

### Dark Mode

Toggle the dark mode switch to invert page colors. Adjust brightness and contrast sliders to your preference. Turn it off to restore the original page appearance.

## Supported Sites

Image URL extraction works on:

- Google Images
- Google Drive
- Google Photos
- Dropbox
- Pinterest
- Most standard websites

## Privacy

All settings are stored locally on your device. No data is sent to external servers.
